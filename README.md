# r-setup
General list of the common R packages that I use all wrapped up into a single script.

The R script contained in this repo will install the most common packages that I use.
* ggplot2
* ggmap
* nnet
* lme4
* lmetest
* knitr
* devtools
* stargazer
* RPostgreSQL
* RMySQL
* roxygen
* rgeos
* rgdal
* maps
* maptools
* xtable
* Hmisc
